# Page romantique "Pour toi, mon amour"

Fichiers créés :


- `index.html` — page web principale (la carte). Ne contient plus d'option d'édition en ligne.
- `styles.css` — styles et décor.
- `script.js` — actions : Imprimer / Télécharger le texte.

Nouveautés visuelles :

- Coeurs animés en arrière-plan.
- Petit portrait circulaire décoratif.
- Titres avec texte en dégradé et animation d'apparition.
Note : l'option d'édition a été retirée. Pour accéder à la carte, ouvrez d'abord la page `surprise.html` et cliquez sur "SURPRISE".

Comment l'ouvrir :

1. Double-cliquez sur `surprise.html` (ou clic droit > Ouvrir avec > votre navigateur) pour l'ouvrir dans Chrome, Edge ou Firefox. Clique sur le bouton "SURPRISE" pour accéder à la carte `index.html`.
2. Pour générer un PDF : cliquez sur le bouton "Imprimer / Enregistrer en PDF" puis choisissez "Enregistrer en PDF" dans la boîte d'impression.

Personnalisation : dites-moi le message exact, la signature, les couleurs, ou si vous souhaitez ajouter une chanson ou une photo — je modifie la page.

Si vous voulez, je peux aussi :
- Générer automatiquement un fichier PDF prêt à imprimer depuis le serveur (nécessite un petit script côté serveur).
- Produire une version MP4 animée (carte animée) ou une carte imprimable A5.
